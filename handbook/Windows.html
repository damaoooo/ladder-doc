    # Windows 用法

    # 安装

    打开安装包，选择中文简体，然后狂点下一步就是了

    # 导入订阅连接

    安装好后你会看到以下界面，点击旁边的订阅

    输入订阅链接，然后点击导入

    # 选择最优节点

    点击该配置进行激活，随后来到代理界面

    进行测速，选择一个延迟最低的节点

    你也可以选择自动选择，如果选择该选项， clash verge rev将会自动测速，并自动选择延迟最低的节点

    # [可选]测试节点可用性

    到测试 页面，点击测试全部， 验证节点是否可用

    # [可选] 查看剩余流量

    在订阅界面，对着你的订阅，右键，首页，即可进入统计界面

    每个节点后面有每个月的流量限制，只需要看上传流量是否超过此限制即可

    # 【重要！！！】开启上网

    到设置里面，开启系统代理。如果有需要，推荐开启开机自启和静默启动.

    如果仍然无法科学上网，请

    1. 彻底关闭clash verge rev
    2. 使用管理员权限运行 clash verge rev, 此时绝大部分问题都能得到解决
    3. 如果还是不行，请私聊我
