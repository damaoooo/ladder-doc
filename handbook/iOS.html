# iPhone 和 iPad

# iPhone 和 iPad

现在我终于有iPhone 和 iPad了

iPhone和iPad 比较麻烦，得搞一个非国区的Apple ID。怎么搞就有来头了，不需要一定是自己的，借用一下别人的也可以，需要安装一个叫做shadowrocket的软件。如果无法使用，请更新到最新版本

# 换App store的Apple ID

进入App Store

点你的头像

往下面一直翻

在最后有一个sign out退出登录

翻回来，重新登陆非国区 的Apple ID

搜索APP 名字，第一次安装需要购买，一次购买，可以安装到多台设备

只有红框的是真的，其余的是广告，安装好之后App长这样

# 手动导入配置

点这里，切换为subscribe/订阅

填写URL（就是一开始的两个网址）和允许不安全打开，最后别忘保存

# 开始使用

使用连通性测试，没啥问题之后，选择一个延迟最低的节点，直接开始使用
