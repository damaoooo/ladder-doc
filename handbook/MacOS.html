    # 安装

    老款intel芯片的mac下载x64.dmg

    M1芯片以及以后的mac下载aarch64.dmg

    把Clash Verge拖进Applications里面，就可以关闭这个窗口了

    成功执行后，按下command + 空格, 输入clash verge，即可运行

    其余的设置和windows设置一模一样。

    强烈建议开启 开机自启 + 静默启动 + 系统代理， 不然很容易找不到。
